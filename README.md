# Onboarding Flow & Interactive Prototype

##  Project Overview

This project was created as **Task 3** during my **UI/UX Design Internship at SyntecxHub**.

The objective was to design a modern and user-friendly onboarding experience using **Figma**, featuring interactive prototypes, smooth animations, reusable components, and developer handoff documentation.

---

##  Features

* Modern onboarding UI
* 5 onboarding screens
* Welcome screen
* Permission request screen
* Personalization screen
* Account setup screen
* Success/Get Started screen
* Interactive prototype
* Smart Animate transitions
* Interactive components
* Conditional button states
* Auto Layout
* Developer handoff notes
